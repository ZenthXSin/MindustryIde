# 节点自动布局系统

## 功能概述

为 Mindustry IDE Web 项目的画布编辑器添加了智能的节点自动布局功能，在添加新字段时自动计算最佳位置，避免节点重叠并保持美观的布局。

## 核心特性

### 1. **智能力导向布局算法** (`calculateAutoLayoutPosition`)

当添加新节点时，系统会：
- 分析现有节点的位置分布
- 生成多个候选位置（围绕现有节点的四周和环形扩展）
- 评估每个位置的得分（考虑重叠、距离、中心偏好）
- 选择最优位置放置新节点

### 2. **候选位置生成** (`generateCandidatePositions`)

系统在以下位置生成候选点：
- **相邻位置**：每个现有节点的上、下、左、右、左下、右下
- **环形扩展**：从原点向外扩展的 3 层环形位置
- 确保有足够的备选位置供选择

### 3. **位置评估函数** (`evaluatePosition`)

评分机制考虑：
- **重叠惩罚**：如果与现有节点重叠，给予极高惩罚 (10000)
- **距离惩罚**：距离越近惩罚越大（反比关系）
- **中心偏好**：轻微偏好靠近中心的区域（权重 0.1）

### 4. **后备网格布局** (`fallbackGridLayout`)

当智能算法无法找到合适位置时，使用网格布局作为后备方案：
- 动态计算列数（最多 5 列）
- 居中对齐的网格排列
- 保持合理的间距

## 用户交互

### 重置视角按钮（集成自动布局）

右下角的 **"↺ 重置视图和自动布局"** 按钮同时执行以下操作：
- 重置画布缩放至 100%
- 重置画布平移至中心位置
- 清除所有手动定位记录
- 对所有节点应用智能自动布局

**触发方式**：
1. 点击右下角的 "↺" 按钮
2. 双击画布空白区域
3. 点击缩放百分比标签（100%）
4. 按下键盘快捷键 `0` 或 `Escape`

### 保留手动调整

- 用户可以拖拽节点到任意位置
- 手动调整的节点会被标记，不会被新节点的自动布局覆盖
- 只有点击“重置视图”时才会清除所有手动定位

## 技术实现

### 关键参数

```javascript
const cardWidth = 260;      // 卡片宽度
const cardHeight = 80;      // 卡片高度
const spacingX = 30;        // 水平间距
const spacingY = 25;        // 垂直间距
const minDistance = 20;     // 最小允许距离
```

### 数据结构

```javascript
// 存储所有节点位置
let cardPositions = new Map();  // fieldName -> {x, y}

// 标记手动定位的节点
let manuallyPositioned = new Set();  // fieldName
```

### 工作流程

1. **添加新字段** → 调用 `addField()`
2. **刷新数据** → 调用 `fetchFields()`
3. **渲染画布** → 调用 `renderCanvasCards()`
4. **位置决策**：
   - 如果已手动定位 → 使用保存的位置
   - 如果已有自动位置 → 复用该位置
   - 如果是全新节点 → 调用 `calculateAutoLayoutPosition()`

## 使用场景

### 场景 1：逐个添加字段
- 每次点击 "+" 添加字段
- 新字段自动出现在最佳位置
- 不会与现有字段重叠

### 场景 2：批量整理
- 手动拖拽导致布局混乱
- 点击右下角 "↺" 按钮或双击画布
- 所有节点重新智能排列，视图重置

### 场景 3：混合模式
- 部分节点手动定位（重要字段）
- 部分节点自动布局（普通字段）
- 系统尊重用户的手动调整

## 优势

✅ **零重叠**：智能检测避免节点重叠  
✅ **美观布局**：优先紧凑且均匀的排列  
✅ **灵活控制**：支持手动调整和自动布局混合  
✅ **性能优秀**：O(n²) 复杂度，适合数百个节点  
✅ **渐进增强**：不影响现有功能，纯新增特性  

## 未来优化方向

1. **力导向动画**：添加平滑的过渡动画
2. **分组布局**：按字段类型分组排列
3. **连接线**：显示字段间的依赖关系
4. **布局预设**：提供多种布局风格（树形、环形、层级等）
5. **撤销/重做**：支持布局操作的撤销
